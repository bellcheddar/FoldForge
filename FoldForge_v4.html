FoldForge — Boltz-2 Prediction Interface

FoldForge provides a browser-based GUI for Boltz-2 structure prediction. Configure inputs, queue jobs, write run scripts to disk, and track results — all in one place.

Workflow Overview

Step 01
Setup Installation
Set the path to your local Boltz-2 binary and save it to Firebase. One-time per machine.
Step 02
Build YAML Input
Enter protein FASTA sequences and ligand SMILES. Add pocket constraints and affinity settings. Export your input.yaml.
Step 03
Configure & Queue Job
Select your YAML, set Boltz-2 run parameters, pick a run directory, then queue and write your run_boltz.zsh script to disk.
Step 04
Analyse Results
Run analyze_boltz.py to generate CSV output. Upload to the Results tab to view confidence scores, pIC50, and affinity data.

Example input.yaml

# FoldForge / Boltz-2 input YAML — generated via Setup tab sequences: - protein: id: GLR sequence: ATVSLWETVQKWREYRRQCQRSLTEDPPPATDLFC... - protein: id: GSA sequence: MGCLGNSKTEDQRNEEKAQREANKKIEKQLQKDK... - ligand: smiles: 'CC[C@H]1C[C@@H](c2ccc(...)cc2-...)N(...)C1' id: D111 constraints: - pocket: binder: D111 contacts: [[GLR, 353]] properties: - affinity: binder: D111

Key Output Metrics

confidence_score
Range: 0 – 1
Overall model confidence. Values above 0.7 indicate high-quality predictions.
affinity_pred_value1
Units: pKd / ΔG proxy
Predicted binding affinity score. Lower is tighter. Used to compute pIC50.
affinity_probability_binary1
Range: 0 – 1
Probability of binder classification (>0.5 = likely binder).
pIC50
Formula: (6 − val) × 1.364
Derived inhibitory concentration estimate. Higher = more potent. >6 is drug-like range.

Setup & YAML Builder

Boltz-2 Binary Path

Set the path to your local Boltz-2 executable. This is saved to Firebase against your user ID and auto-loaded on next visit.

Example: /opt/anaconda3/envs/boltz_2_2_1_env/bin/boltz

User ID

Your settings (Boltz path, saved YAMLs) are stored in Firebase under this user ID. Change it to switch profiles.
